# Note-App

A simple and elegant Notes Application built using **HTML, CSS, and JavaScript**.

## Features

- Create and save notes instantly
- Delete notes when no longer needed
- Automatic data persistence using Local Storage
- Clean vintage-themed user interface
- Responsive and easy-to-use design
- No backend or database required

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage API
